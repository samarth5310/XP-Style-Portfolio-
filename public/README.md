# Public Assets

## Icons (`/icons/`)
Place your icon files here:
- `windows.png` - Windows XP start button logo (20x20)
- `user.png` - About Me window icon
- `folder.png` - Projects window icon
- `file.png` - Resume window icon
- `mail.png` - Contact window icon
- `terminal.png` - Terminal window icon
- `globe.png` - Explorer window icon

## Sounds (`/sounds/`)
Place your sound files here:
- `startup.mp3` - Windows XP startup sound
- `click.mp3` - Button click sound
- `error.mp3` - Error sound
- `close.mp3` - Window close sound

## Wallpapers (`/wallpapers/`)
Optional wallpaper images for the desktop background.

## Resume
- `resume.pdf` - Your downloadable resume (already placed)

## Where to get assets:
- XP Sounds: Search "Windows XP sounds" (startup.wav, etc.)
- Icons: Use any 16x16 or 32x32 PNG icons
- The app works without these files - it uses Lucide React icons as fallbacks in the UI
